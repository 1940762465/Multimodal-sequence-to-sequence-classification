# Multimodal-sequence-to-sequence-classification
Numerical modal analysis of well logging data is the most suitable for exploring the encoder model; Integrate image
modal data and conduct sequence-to-sequence model classification for multiple modalities.

For the division of the training set and test set of the logging data, it is divided into well section division or blind well division. Currently, the attempts to convert the image modes of well logging data mainly fall into four methods, among which the first method is the most common:
1. Direct visualization of multiple image modes of well logging curves through windowing 
2. Wavelet transform converts a single well logging curve into a two-dimensional image
3. Gram converts a single well logging curve into a two-dimensional image mode
4. Direct two-dimensional visualization of well logging curves
